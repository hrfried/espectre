## Validation rule

- labels are coherent (`baseline` vs `movement`)
- `movement_variance > baseline_variance` (ratio >= 1.5x)

- `Baseline Var`: variance of spatial turbulence on baseline file

- `Ratio`: `Movement Var / Baseline Var`
